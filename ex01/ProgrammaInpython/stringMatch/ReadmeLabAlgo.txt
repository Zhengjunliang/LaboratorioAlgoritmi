## Guida all'Esecuzione del Programma per l'Analisi delle Prestazioni degli Algoritmi di Ricerca di Stringhe

- Python 3.x installato sul sistema.
- Librerie Python aggiuntive: matplotlib (per la creazione di grafici).

    ```
    python --version
    ```

    ```
    pip install matplotlib
    ```
